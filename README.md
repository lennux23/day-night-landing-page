# Usage

You need [git](https://git-scm.com/) and [node.js](https://nodejs.org/) on your computer before running.

1. `git clone https://github.com/lennux23/day-night-landing-page`
2. `cd day-night-landing-page && rm -rf .git`
3. `npm install`
4. `npm start`
